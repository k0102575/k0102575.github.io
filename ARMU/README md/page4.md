# 4. 일반모드 - 이벤트 등록 / 조회 / 수정 / 삭제

## 페이지 소개
* 일반모드에서 일반회원이 이벤트를 등록하는 화면 입니다.
* 등록한 이벤트의 내용을  확인하고, 수정 및 삭제가 가능합니다
<br><br>

## 구동원리 
* 다음버튼을 누르면서 단계별로 페이지 이동이 일어나는것처럼 표시합니다.
* 프로그래스 바를 통해 등록 진행상태 표시합니다.
* 이벤트 조회시 참여요청 뮤지션 과 이벤트에 지원한 뮤지션 조회 가능합니다.
* 이벤트 등록 및 편집시 필수항목 미 입력 및 미 체크시 알림창 표시합니다.
* 이벤트 편집/삭제 시 해당 이벤트에 걸려 있는 모든 뮤지션의 지원을 비활성화 처리합니다.
<br><br>
