# 10. 뮤지션모드 - 뮤지션 회원가입

## 페이지 소개
* 일반회원이 뮤지션회원으로 회원가입 하는 페이지 입니다.
<br><br>

## 구동원리 
* 다음버튼을 누르면서 단계별로 페이지 이동이 일어나는것처럼 표시합니다
* 프로그래스 바를 통해 등록 진행상태 표시합니다
* 필수사항 미입력 및 미체크시 알림창으로 알려줍니다.
<br><br>
