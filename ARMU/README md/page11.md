# 11. 뮤지션모드 - 뮤지션 포트폴리오 관리

## 페이지 소개
* 뮤지션이 자신의 경력 및 정보를 추가/변경/삭제하여 자신의 포트폴리오를 관리하는 화면입니다.
<br><br>

## 구동원리 
* 경력사진을 업로드 할 때 미리보기 이미지를 생성합니다.
* 사진 업로드시 서버에서 썸네일 처리 합니다.
* 경력 0개 입력시 및 유튜브 URL정보가 틀릴시 알림창 처리 합니다.
* 사용자로 부터 입력 받은 유튜브 URL 정보에서 동영상 ID값을 추출하여 iframe 태그를 이용해 동영상 출력 합니다.
<br><br>
