# 14. 뮤지션모드 - 매칭된 이벤트 조회

## 페이지 소개
* 뮤지션 회원이 이미 매칭된 이벤트(진행중,완료) 목록을 조회하는 화면입니다.

## 구동원리 
* 매칭된 이벤트 목록을 진행상태(진행중/완료)에 따라 탭으로 구분하여, 다른형식의 UI로 목록을 보여줍니다.
