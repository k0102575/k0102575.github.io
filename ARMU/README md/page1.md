# 1. 회원가입 / 로그인, 메뉴, 프로필 관리

## 페이지 소개
* 일반모드 회원가입 페이지 및 로그인 페이지입니다.
* 뮤지션 회원의 닉네임, 모든회원의 사진,비밀번호를 변경하는 개인정보 관리 화면입니다.
<br><br>

## 구동원리 
* 회원가입을 할 때 프로필 사진 업로드 시 미리보기 이미지 생성합니다.
* 사진 업로드시 서버에서 썸네일 처리합니다.
* 로그인시 DB의 정보와 맞지 않을시 로그인 실패 알림창 처리합니다.
* 필수 항목 미 입력시 표시 및 회원가입 미처리 합니다.
* DB에서 받아온 암호화된 비밀번호와 사용자가 입력한 암호 비교 확인합니다.
<br><br>
